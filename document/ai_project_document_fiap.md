# Documento do Projeto

## Problema escolhido

O projeto trata o monitoramento operacional da colheita de cana-de-acucar, com foco em registrar resultados por fazenda, talhao e safra para apoiar analises de perda e tomada de decisao.

## Solucao proposta

Foi desenvolvida uma aplicacao web com:

- front-end em Next.js + React;
- API em Python com FastAPI;
- persistencia principal em Oracle;
- exportacao complementar em JSON e TXT.

## Requisitos da disciplina contemplados

- funcoes e procedimentos com passagem de parametros;
- estruturas de dados para organizacao de entidades e respostas;
- manipulacao de arquivos JSON e TXT;
- conexao real com banco Oracle.

## Recorte do MVP

- CRUD de fazendas;
- CRUD de talhoes;
- CRUD de registros de colheita;
- relatorios consolidados por safra, fazenda e talhao;
- dashboard operacional com indicadores principais.
